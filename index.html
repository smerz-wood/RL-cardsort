Library website card sort

Choose one deck, organize its cards into categories that make sense to you, name those categories, and download your completed result as a CSV file.

Saving is device-specific. One draft is stored for each deck in this browser. Nothing is uploaded to GitHub automatically.

Participant instructions

  1. Choose the assigned deck and enter your participant code.
  2. Create and name categories in your own words.
  3. Drag every card into the category where you would expect to find it.
  4. Select Save draft on this device whenever needed.
  5. Select Download completed CSV and return the downloaded file to the study coordinator.

The CSV contains one readable row per card. Cards left in the yellow uncategorized column are clearly marked as uncategorized.
